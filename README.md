# BattleList
74eo-BattleList

## 说明
因为未发现74有什么好用的战斗列表，且自己希望有每场的详细数据，所以写了本插件。
本来准备使用json存储，但是考虑到效率问题，改用sqlite，测试20W行记录的读取大约在4秒左右，完全可满足舰娘大量的的战绩存储。

## 使用方法
下载后将所有文件释放到74目录下的plugins目录中，重新开启74浏览器即可。
